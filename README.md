# MT-tools-nanopore
Description<br>

Collection of python programs for data analysis of output generated by the program in repository nanopore-3D.
Warning: none of the program consider the reduce volume for volume cell element that are within the membrane.

### convert_vtk_to_numpy_vista.py:

Converts vtk file to numpy array or txt array and saves to file. Uses pyvista to read vtk file. <br>
Usage: python convert_vtk_to_numpy_vista.py <input_vtk_file>  npy or txt

### make_avfdis.py:

Computes the global average degree of dissociation. <br>

### make_pH_slices_nanopore.py:

Computes pH and plots slice of 3D data.

positional arguments:<br>
  filename              Input filename  : systemfile name. <br>

options:<br>
  -h, --help            show this help message and exit <br>
  --filetype FILETYPE   Type of input file: vtk or dat <br>
  --plane PLANE         Select plane of slice (default z) : x, y or z <br>
  --coordplane COORDPLANE
                        Select coordinate plane of slice (default middle) : number <br>



### make_pH_slices_planar.py:

Identical to make_slices_vtk.py. Applicable to 3D planar brush calculation. 
Can potentially compute also ion distribution of different monovalent and divalent ions (Na,K,Cl,Mg,Ca) compared to  make_pH_slices_nanopore.py 
that can only consider Na and Cl.<br>

### make_pH_slices.py:

Similar to make_pH_slices_nanopore.py

### make_pH.py:

Computes the local, position dependent pH fields. For all avsol*.vtk and poten*.vtk found in local directory the local pH is computed. Output in vtk format.

### make_slices_vtk.py:

Computes and plots slice of 3D data. Only for vtk data input files.<br>

positional arguments:<br>
  filename              Input filename<br>

options:<br>
  -h, --help            show this help message and exit<br>
  --filetype FILETYPE   Type of input file: vtk or dat<br>
  --plane PLANE         Select plane of slice (default z)<br>
  --coordplane COORDPLANE
                        Select coordinate plane of slice (default middle)<br>

### make_slices.py:

Identical to make_slices_vtk.py both works for vtk and dat input files.<br>

positional arguments:<br>
  filename              Input filename<br>

options:<br>
  -h, --help            show this help message and exit<br>
  --filetype FILETYPE   Type of input file: vtk or dat<br>
  --plane PLANE         Select plane of slice (default z)<br>
  --coordplane COORDPLANE
                        Select coordinate plane of slice (default middle)<br>



### Requirement  

matplotlib, numpy, pyvista, matplotlib, argparse
